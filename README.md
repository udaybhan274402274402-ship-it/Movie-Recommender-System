# 🎬 Movie Recommender System

A content-based Movie Recommender System built using **Python, Pandas, Scikit-learn, Streamlit, and TMDB API**. This application recommends movies similar to the one selected by the user and displays their posters using the TMDB API.

---

## 🚀 Features

* Recommend top 5 similar movies.
* Display movie posters using TMDB API.
* Interactive Streamlit web interface.
* Fast movie search through dropdown selection.
* Content-Based Recommendation System using Cosine Similarity.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Movie-Recommender-System.git
```

Move into the project directory:

```bash
cd Movie-Recommender-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
